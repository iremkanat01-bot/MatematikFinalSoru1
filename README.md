Ad Soyad: İrem KANAT

Matematik Final – 1. Soru

Bu projede matematik final sınavı 1. sorusunun çözümü yer almaktadır.

Proje Açıklaması

Bu program, Python dili kullanılarak ikinci dereceden bir bilinmeyenli denklemlerin (ax^2 + bx + c = 0) köklerini bulmak amacıyla geliştirilmiştir.

Programın Özellikleri:

Kullanıcıdan a, b ve c katsayılarını alır.
Diskriminant (Delta) hesaplaması yapar.
Delta değerine göre;
İki farklı reel kök varsa hesaplar ve yazdırır.
Çakışık (tek) kök varsa hesaplar ve yazdırır.
Reel kök yoksa kullanıcıyı bilgilendirir.
